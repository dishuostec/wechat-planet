# 微信运营平台

## 设计方案

* 控制台功能抽象为REST风格API，与前端展示分离
* 控制台划分为独立的模块，每个模块包含API和服务两部分

## API

* 【API资源】 /console/api/**MODULE**/**RESOURCE**

* 【前端模版】 /console/tpl/**MODULE**/**VIEW**

* 【模块服务】 /public/**MODULE**/**SERVICE**
